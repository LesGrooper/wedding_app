Letakkan file musik latar di folder ini dengan nama:

    bgm.mp3

Path ini akan tersedia di aplikasi sebagai: /assets/audio/bgm.mp3
(folder `public/` dipetakan ke root oleh angular.json)

Tips:
- Format: .mp3 (paling kompatibel di semua browser & iOS Safari).
- Ukuran wajar: < 3-4 MB agar cepat dimuat di HP.
- Musik mulai otomatis saat tamu menekan tombol "Buka Undangan".
- Jika file ini belum ada, aplikasi tetap berjalan normal (tanpa musik).
